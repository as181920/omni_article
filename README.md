# OmniArticle
通用文章功能(Rails engine)

## Usage

- `GET /omni_article/api-docs`
- `GET /omni_article/api-docs/openapi.json`

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'omni_article', git: "git@github.com:as181920/omni_article.git", branch: "master"
```

And then execute:
```bash
$ bundle
```
